# DevOps LearnIT — AI Context

## Project Purpose

6-month DevOps bootcamp learning environment (March–August 2026). Hub repo containing course exercises, notes, and progress tracking. Part of a multi-repo structure with satellite repos for reusable tools and projects.

## Repository Map

| Repo | Purpose | Location |
|------|---------|----------|
| **DevOps-LearnIT** (this repo) | Course modules, exercises, notes | Hub |
| **devops-toolbox** | Reusable scripts, configs, automation | Satellite |
| **diploma-project** | Final diploma project (created later) | Satellite |

| 11 | Cloud (AWS) | EC2, S3, IAM, VPC, RDS, CloudFormation basics |
| 12 | IaC, K8s & Monitoring | Terraform, Kubernetes, Prometheus, Grafana, ELK stack |

## Conventions

- **Language**: All documentation in English
- **Folder naming**: Kebab-case with numeric prefix (`01-devops-intro-and-os/`)
- **Exercises**: Each module has an `exercises/` subfolder
- **Notes**: Add personal notes to each module's `README.md`
- **Cheatsheets**: Quick-reference files go in `cheatsheets/`
- **Secrets**: Never commit `.env`, keys, or credentials — `.gitignore` handles this
- **Commits**: Use conventional style (`feat:`, `fix:`, `docs:`, `chore:`)
